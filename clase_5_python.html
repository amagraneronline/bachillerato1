Python: Listas Pro

Gestiona colecciones de datos como un desarrollador experto

1. Estructura y Visualización

Las listas permiten guardar múltiples elementos en una sola variable. Para mostrar elementos, podemos usar el índice o recorrerlas.

mochila = ["Poción", "Antídoto", "Baya"]

# Mostrar lista completa
print(mochila) 

# Mostrar mediante bucle (más profesional)
for objeto in mochila:
    print(f"Tienes: {objeto}")
Nota: Los índices en Python comienzan siempre en 0.

2. Añadir Elementos

Existen dos formas principales de ampliar tu lista: al final o en una posición específica.

inventario = ["Mapa"]

# Añadir al final con append()
inventario.append("Brújula")

# Insertar en posición específica (índice, valor)
inventario.insert(0, "Linterna") 

# Resultado: ["Linterna", "Mapa", "Brújula"]

3. Eliminar Elementos

Puedes eliminar por nombre del objeto o por su posición en la lista.

equipo = ["Pikachu", "Charizard", "Squirtle"]

# Eliminar por nombre
equipo.remove("Charizard")

# Eliminar por índice y obtener el valor
borrado = equipo.pop(1) 

print(f"Se eliminó a: {borrado}")
🚨 Error común: Intentar usar remove() con un elemento que no existe lanzará un error.

4. Buscar y Contar

¿Está el objeto en la lista? ¿Cuántos hay? Python lo hace muy sencillo.

premios = ["Oro", "Plata", "Oro", "Bronce"]

# Buscar si existe (devuelve True/False)
existe = "Oro" in premios

# Contar cuántas veces aparece
cantidad = premios.count("Oro")

# Encontrar la posición (índice)
indice = premios.index("Plata")

5. Ordenar y Extremos

Acceder al primer/último elemento y organizar la lista alfabética o numéricamente.

notas = [8, 10, 5, 9]

# Mostrar primero y último
print(f"Primero: {notas[0]} | Último: {notas[-1]}")

# Ordenar de menor a mayor
notas.sort()

# Ordenar de mayor a menor
notas.sort(reverse=True)

6. Vaciar la Lista

Cuando necesitas reiniciar una colección por completo sin borrar la variable.

notificaciones = ["Msg 1", "Msg 2", "Msg 3"]

# Vaciar todos los elementos
notificaciones.clear()

print(len(notificaciones)) # Imprime 0
Pro Tip: len(lista) te devuelve siempre el número total de elementos.

7. Proyecto: Gestor de Tareas

Aplicando todo lo aprendido en un script funcional.

tareas = ["Estudiar", "Gimnasio"]

while True:
    print(f"\nPENDIENTES ({len(tareas)}): {tareas}")
    op = input("1. Añadir | 2. Completar | 3. Ordenar | 4. Salir: ")

    match op:
        case "1":
            nueva = input("Tarea: ")
            tareas.append(nueva)
        case "2":
            if tareas:
                tareas.pop(0) # Elimina la primera
        case "3":
            tareas.sort()
        case "4":
            break
