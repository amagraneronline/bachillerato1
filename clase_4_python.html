Python: Estructuras Repetitivas

Aprende a dominar el tiempo y la lógica de tus programas

1. ¿Qué son los bucles?

Un bucle permite repetir un bloque de código automáticamente. Es la clave para que tus programas sean eficientes.

Analogía: En lugar de decirle a un Pokémon "entrena" 10 veces, usas un bucle que diga: "Mientras no hayas terminado las 10 vueltas, sigue corriendo".

2. El bucle for

Úsalo cuando sepas exactamente cuántas veces quieres repetir algo o para recorrer una lista.

equipo = ["Pikachu", "Eevee", "Ditto"]

for pokemon in equipo:
    print(f"¡{pokemon}, yo te elijo!")

3. La función range()

Genera números en un rango. Recuerda: el límite final no se incluye.

for i in range(1, 4):
    print(f"Intento {i}...")
# Imprimirá: Intento 1, Intento 2, Intento 3

4. El "Paso" en range()

Podemos saltar números o incluso contar hacia atrás.

# Contar de 10 en 10 hasta 50
for n in range(0, 51, 10):
    print(n)
Reto: Para múltiplos de 5 hasta el 50, usa range(0, 51, 5).

5. El bucle while

Se repite mientras la condición sea True. Es ideal cuando no sabemos el final exacto.

energia = 30
while energia > 0:
    print(f"Entrenando... Energía: {energia}")
    energia -= 10 # Reducimos la variable de control

6. Variables de Control

Si la condición nunca se vuelve False, el bucle será infinito.

🚨 ¡Cuidado! Si no cambias el valor de la variable dentro del while, el programa se bloqueará.

7. Proyecto: Menú Infinito

Combinamos while y match para crear un programa que no se cierra hasta que el usuario quiera.

jugando = True

while jugando:
    print("\n--- CENTRO POKÉMON ---")
    print("1. Curar equipo")
    print("2. Ver estadísticas")
    print("3. Salir")
    
    opcion = input("Elige una opción: ")

    match opcion:
        case "1":
            print("¡Equipo curado al 100%! ✨")
        case "2":
            print("Nivel: 25 | Medallas: 2")
        case "3":
            print("¡Adiós, entrenador! 👋")
            jugando = False # ¡ESTO ROMPE EL BUCLE!
        case _:
            print("Opción inválida.")
