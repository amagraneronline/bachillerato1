1. Entrada y Conversión (Casting)

El programa necesita "escuchar" al usuario. Recuerda: input() siempre devuelve texto.

# Capturando la esencia de un Pokémon
nombre = input("Nombre: ") 
nivel = int(input("Nivel: "))    # Convertimos a Entero
peso = float(input("Peso: "))   # Convertimos a Decimal

print(f"{nombre} está listo para luchar.")
¿Por qué convertir?
Sin int(), si intentas hacer nivel + 1, Python intentará sumar "Texto" + "Número" y el programa fallará.
Tip de Clase:
Usa siempre float() para datos precisos como altura, peso o multiplicadores de daño.

4. Operadores Lógicos

Combinamos varias preguntas para tomar una decisión estratégica.

and (Y)

Verdadero solo si TODO se cumple.

puedo_evolucionar = nivel >= 20 and tiene_objeto == True

or (O)

Verdadero si AL MENOS UNA cosa se cumple.

ganas_combate = enemigo_hp <= 0 or enemigo_huye == True

not (NO)

Invierte el resultado (lo que es True pasa a False).
