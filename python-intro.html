Python usa las palabras clave if, elif (else if) y else para definir condicionales. Los bloques de c√≥digo anidados se indican con la tecla tabulador (TAB).

Estructuras repetitivas

Los bucles (loops) permiten repetir un bloque de c√≥digo m√∫ltiples veces. En Python tenemos dos tipos: for y while.

El bucle for se usa cuando sabemos exactamente cu√°ntas veces queremos repetir algo. Se combina frecuentemente con range().

Funciones y m√≥dulos

Las funciones son bloques de c√≥digo reutilizables que realizan una tarea espec√≠fica. Los m√≥dulos agrupan funciones en archivos separados.

Se definen con la palabra clave def, seguida del nombre y par√©ntesis. Pueden recibir par√°metros y devolver valores con return.

Estructura y ejemplos de funciones
# Esquema general def nombre_de_la_funcion(parametro1, parametro2): # bloque de c√≥digo return valor_de_retorno # Funci√≥n sin return ‚Äî solo ejecuta acciones def saludar(nombre): print(f"Hola, {nombre}") saludar("Ana") # Hola, Ana saludar("Luis") # Hola, Luis # Funci√≥n con return ‚Äî calcula y devuelve un valor def sumar(a, b): return a + b resultado = sumar(5, 3) print(resultado) # 8

Flujo de ejecuci√≥n

  • El ordenador lee la definici√≥n de la funci√≥n pero no la ejecuta hasta que sea llamada.
  • Al llamar a sumar(5, 3), Python busca la definici√≥n y sustituye: a=5, b=3.
  • Se ejecuta el cuerpo de la funci√≥n: return 5+3.
  • La funci√≥n devuelve 8 al punto donde fue llamada y el programa contin√∫a.

El alcance (scope) de una variable determina d√≥nde puede usarse dentro del c√≥digo.
