¿Qué es Python?

Un lenguaje de programación interpretado, legible y muy usado en el mundo real.

  • Ideal para empezar a programar.
  • Usado en ciencia de datos, web, automatización, IA…
  • Gran comunidad y mucha documentación.
Pregunta al grupo: ¿Dónde habéis visto o escuchado Python antes?

¿Dónde escribimos código?

  • En archivos .py ejecutados con python archivo.py.
  • En el intérprete interactivo (REPL).
  • En editores como VS Code, Thonny, PyCharm…
Prueba rápida 
print("Hola, Python")
Actividad: que cada persona escriba y ejecute este código en su entorno.

Qué es una variable
Concepto básico

Idea clave

Una variable es un nombre que guarda un valor en memoria.

Ejemplo 
edad = 25
nombre = "Ana"
altura = 1.68
es_estudiante = True
