Python Nivel 0: Variables y Datos

Aprende a guardar información y mostrarla al mundo

1. ¿Qué es una Variable?

Imagina una variable como una caja con nombre donde guardas un valor usando el signo igual (=).

# El signo "=" no significa "igualdad matemática", significa "GUARDAR"
nombre_entrenador = "Ash"
pokemon_inicial = "Pikachu"
nivel = 5
Reglas de oro:
  • No pueden empezar con números.
  • No pueden tener espacios (usamos el guion bajo _).
  • Python distingue entre mayúsculas y minúsculas.

3. Mostrar Datos: print()

Para unir o "concatenar" valores, podemos usar la coma , o el símbolo +.

nombre = "Bulbasaur"
tipo = "Planta"

# Usando comas (añade un espacio automático)
print("Tu Pokémon es", nombre, "y es tipo", tipo)

# Usando + (solo para texto, NO añade espacio)
print("Pokémon: " + nombre)
Ojo: No puedes usar + para unir un texto con un número (int). Para eso, ¡mejor usa la coma o el f-string!

4. Cadenas de Formato: f-print

Es la forma más moderna y cómoda de escribir mensajes mezclando variables y texto.

nombre = "Charmander"
hp = 39

# Ponemos una 'f' antes de las comillas 
# y las variables entre llaves { }
print(f"El Pokémon {nombre} tiene {hp} puntos de vida.")
Ventaja: Puedes hacer cálculos dentro de las llaves:
print(f"El doble de la vida es: {hp * 2}")

5. Trucos de print: sep y end

Por defecto, print() separa los elementos con un espacio y al final salta a la línea siguiente. ¡Pero podemos cambiarlo!

A. El parámetro sep (Separador)

Define qué carácter aparece entre los elementos que imprimes.

# Ejemplo: Crear una ruta de evolución
print("Pichu", "Pikachu", "Raichu", sep=" -> ")
# Resultado: Pichu -> Pikachu -> Raichu

# Ejemplo: Formato de fecha para un registro
dia, mes, año = 10, 05, 2024
print(dia, mes, año, sep="/")
# Resultado: 10/05/2024

B. El parámetro end (Finalizador)

Define qué se imprime al final. Por defecto es un salto de línea (\n). Si lo cambias, el siguiente print aparecerá pegado al anterior.

# Ejemplo: Barra de carga de salud
print("Salud: [", end="")
print("##########", end="")
print("] 100%")
# Resultado: Salud: [##########] 100% (¡Todo en una sola línea!)
💡 Caso de uso sep=""
Si quieres unir cosas sin ningún espacio entre ellas:
print("Nivel", 50, sep="")
Resultado: Nivel50
💡 Caso de uso end=" "
Úsalo para que varios print se vean como una frase continua en lugar de una lista vertical.
