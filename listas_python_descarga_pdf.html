Preguntas de respuesta corta / código

Pregunta 9

  • Añadir la edad 22 al final de la lista.
  • Modificar el primer elemento para que valga 16.
Pregunta 10

Dada la lista numeros = [3, 1, 4, 1, 5], escribe el código para:

  • Mostrar todos los elementos de la lista por pantalla.
  • Mostrar cuántos elementos tiene la lista.
Pregunta 11

Pregunta 12

Indica si el siguiente código es correcto o incorrecto. Si es incorrecto, explica brevemente por qué.

letras = ["a", "b", "c"]
letras.remove("d")

Pregunta 13

Pregunta 14

  • Ordenar la lista de menor a mayor.
  • Mostrar la lista ya ordenada.
