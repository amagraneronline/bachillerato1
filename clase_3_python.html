Toma de Decisiones en Python

Estructuras If, Elif, Else y Match

1. El Condicional Simple: if

Se usa para ejecutar un bloque de código solo si se cumple una condición.

nivel = 18

if nivel >= 18:
    print("¡Tu Pokémon puede evolucionar!")
    print("Preparando animación...")
⚠️ ¡Atención a la Indentación!
En Python, todo lo que esté "dentro" del IF debe llevar 4 espacios (o un Tabulador). Si no lo haces, Python no sabrá qué código depende de la condición.

2. Si no ocurre... else

Sirve para dar una alternativa cuando la condición no se cumple.

hp = 0

if hp > 0:
    print("¡El Pokémon sigue luchando!")
else:
    print("El Pokémon se ha debilitado...")
    print("Llévalo al Centro Pokémon.")

3. Múltiples opciones: elif

Cuando tienes más de dos caminos posibles. Es una abreviatura de "Else If".

pokemon = "Charmander"

if pokemon == "Charmander":
    print("Tipo Fuego 🔥")
elif pokemon == "Squirtle":
    print("Tipo Agua 💧")
elif pokemon == "Bulbasaur":
    print("Tipo Planta 🌿")
else:
    print("Tipo desconocido ❓")
Python revisa las condiciones de arriba a abajo. En cuanto una es True, ejecuta su código y se salta el resto.

4. El nuevo match

Introducido en Python 3.10, es una forma más limpia de hacer muchos elif cuando comparas una misma variable.

comando = "Atacar"

match comando:
    case "Atacar":
        print("¡Has usado Placaje!")
    case "Pocion":
        print("¡Vida recuperada!")
    case "Huir":
        print("Has escapado sano y salvo.")
    case _:  # Este es el caso por defecto (como el else)
        print("Comando no reconocido.")
¿Por qué usar Match? Es mucho más fácil de leer cuando tienes un menú de opciones o una lista larga de casos.

5. Proyecto: Menú de Combate

Combinamos todo: mostramos un menú, pedimos una opción y decidimos qué hacer.

# 1. Mostramos las opciones
print("--- MENÚ DE COMBATE ---")
print("1. Atacar")
print("2. Usar Poción")
print("3. Huir")

# 2. Pedimos la opción al usuario
opcion = input("Selecciona una opción (1-3): ")

# 3. Gestionamos la decisión con match
match opcion:
    case "1":
        print("¡Pikachu usa Rayo! ⚡")
    case "2":
        print("Has recuperado 20 HP 🧪")
    case "3":
        print("Escapaste del combate... 🏃")
    case _:
        print("Error: Opción no válida.")
💡 Nota importante: Fíjate que en los case usamos "1" (con comillas). Esto es porque input() nos da un String. Si quisiéramos usar números sin comillas, tendríamos que haber hecho int(input(...)) antes.
